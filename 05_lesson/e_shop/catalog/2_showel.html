Лопата снеговая

Краткое описание товара

Прочная, надежная лопата для снегокопания

Характеристики товара

Длина: 150 сантиметров
Вес: 800 грамм
Материал: Пластик
Грузоподъемность: 80 килограмм

Подробное описание товара

Прочная удобная лопата для всех видов снежных работ. Эргономическая ручка и усиленное лезвие позволяют копать снег не хуже армейского бульдозера. Увеличенный ковш берет на 30% больше снега по сравнению с другими моделями снеговых лопат. Незаменимый инструмент для откапывания машины по утрам, строительства снежной крепости, расчистки садового участка.

Важное предупреждение!

Внимание, не предназначена для аварийно-спасательных работ!
